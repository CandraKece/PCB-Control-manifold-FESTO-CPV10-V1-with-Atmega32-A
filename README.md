# PCB-Control-manifold-FESTO-CPV10-V1-with-Atmega32-A
- It integrates multiple solenoid valves and control with PCBs into a single assembly
- Communication with I2c
- uploading with SPI 
- Can using 24 volt and 5 volt for operation (note : only using 24 volt if you want to control valve or something that's operate with 24 volt)
- Not only valve but you can using this PCB to control another thing same thing as relay but without coil
(coution this PCB only maximum at 0,5 A if you using above the maximum ampere the transistor can get hot and burning)
